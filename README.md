How to run the file:

Step 1: Download the code from the Code folder
Step 2: Choose a dataset with 1 or 4 features and download it from the Dataset folder (Remember to put it in the same folder as the code)
Note: If you want to change the dataset, simply replace the one that you have already downloaded with the new one
Step 3: Open the code file with Visual Studio Code (or Jupiter notebook, Colab, etc), then click Run all
